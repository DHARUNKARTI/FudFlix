# 🍽️ FudFlix

A modern, visually rich landing page for a campus-focused food discovery platform.

## 🚀 Overview

FudFlix is a UI-focused landing page designed to showcase a curated food experience for university students.  
The design emphasizes clean layout, layered visuals, and modern typography.

---

## ✨ Features

- 🎨 Gradient background with warm tones
- 🧱 Layered card UI (image + floating content)
- 🔄 Overlapping layout using CSS positioning
- 📱 Responsive-centered design
- 🎯 Clean typography and spacing
- 🧩 Component-based structure (HTML + CSS)

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Positioning, Gradients)
- No frameworks (pure CSS implementation)
